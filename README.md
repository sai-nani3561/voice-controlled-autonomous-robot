# Voice-Controlled and Autonomous Mobile Robot with Safety-Priority Navigation

## Overview

This project was developed as part of an MSc Dissertation in Robotics and Embedded Systems.

The robot supports three operating modes:

* Manual Control
* Voice-Controlled Navigation
* Autonomous Navigation

A safety-priority architecture ensures that cliff detection and emergency-stop mechanisms override all user commands.

---

## Features

### Manual Mode

* Web-based robot control
* Touch joystick control
* Real-time speed adjustment

### Voice Control Mode

* Wake-word activation ("Hello")
* Voice command recognition
* Speech feedback responses
* Mode switching through voice

### Autonomous Mode

* Obstacle avoidance
* Direction scanning
* Automatic path selection

### Safety System

* Cliff detection using grayscale sensors
* Emergency stop functionality
* Safety-priority control logic

### Monitoring and Logging

* Real-time distance monitoring
* CSV telemetry logging
* Timestamped event tracking
